Конструктор градусов, все параметры должны иметь положительные значения, прочая корректность проверяется в соответствии с шаблонным типом.

Аргументы
degreesцелое, положительное число градусов
minutesцелое, положительное число минут не превосходящее 59
secondsцелое, положительное число секунд не превосходящее 59
millisecondsцелое, положительное число милисекунд не превосходящее 1000
is_negativeфлаг отрицательных градусов, true значения будут приниматься как отрицательные, по-умолчанию имеет значение false положительные
